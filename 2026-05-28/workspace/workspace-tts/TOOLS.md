# TOOLS.md - 本地笔记

Skills 定义工具_如何_工作。这个文件放你的_细节_ —— 属于你独特设置的东西。

## 这里放什么

比如：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好声音
- 音箱/房间名称
- 设备昵称
- 任何环境特定的配置

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，动作触发

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好声音："Nova"（温暖，略英式）
- 默认音箱：厨房 HomePod
```

## 为什么分开？

Skills 是共享的。你的配置是你的。分开存放意味着你可以更新 skills 而不丢失你的笔记，也可以分享 skills 而不泄露你的基础设施。
