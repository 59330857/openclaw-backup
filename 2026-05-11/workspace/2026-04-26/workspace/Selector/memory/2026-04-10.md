# 2026-04-10 晚间学习记录

## 任务
HALEI让我研究 clawhub.1688.com 上的Skills，筛选适合TikTok东南亚发饰选品的工具。

## 核心发现

### 🏆 直接推荐安装（选品流程强相关）

| Skill | 用途 | 说明 |
|-------|------|------|
| **douyin-tiktok-trends** | TikTok/抖音趋势抓取 | 获取热搜话题、标签、达人 – 可替代部分数据采集工作 |
| **ecommerce-product-picker** | 跨境电商选品工具 | 实时趋势+TikTok热度+利润计算，支持任意类目 |
| **1688-product-search** | 1688商品搜索API | 关键词+图片搜索+商品详情，官方API支持多语言 |
| **taobao-image-search** | 淘宝以图搜款 | 可作为1688以图搜款的备选/补充方案 |

### 🔧 可考虑（辅助功能）

| Skill | 用途 | 说明 |
|-------|------|------|
| **ecommerce-product-research** | TikTok/Facebook广告数据选品 | 基于PipiAds数据，需验证数据源是否覆盖东南亚 |
| **1688-sourcing** | 1688采购+利润计算 | 辅助1688商品调研和利润测算 |
| **query-1688-product-detail** | 1688商品详情查询 | API方式拉取商品详情（需AlphaShop API） |
| **ecommerce-analyzer** | 电商数据分析监控 | 支持国内平台价格/评论/关键词监控 |

### ❌ 暂不推荐（不适用）

| Skill | 原因 |
|-------|------|
| 1688-to-ozon | 针对OZON平台，不适合TikTok东南亚 |
| taobao-shopping | 偏向购物决策辅助，非选品用途 |
| cross-border-ecommerce-skill | 偏向市场准入/物流合规，非选品数据 |

## 选品流程映射

现有流程：
1. 数据网站获取商品数据 → `ecommerce-product-picker` 或 `douyin-tiktok-trends`
2. 1688以图搜款 → `1688-product-search` 或 `taobao-image-search`
3. 采集链接/价格/款式 → `1688-product-search` 或 `query-1688-product-detail`
4. 录入飞书表格 → 现有流程不变
