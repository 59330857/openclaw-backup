## 利润测算公式

### 越南站（v3.0 更新）
汇率：3570 VND/USD
平台费率：佣金13% + 交易费5% + VAT12% + 提现1% + 达人10% = 41% + ¥2固定

```
售价_RMB = 售价_VND / 3570
总平台成本 = 售价_RMB × 41% + 2
真实毛利润 = 售价_RMB - 进货价 - 物流费 - 总平台成本
真实毛利率 = 真实毛利润 / 售价_RMB
毛利率要求：≥ 20%（套装款按套装毛利率，应≥40%）
```

### 泰国站
```
售价 (THB) = (1688采购价CNY + 头程运费CNY) × 5.0 × 1.25
毛利率要求：≥ 25%
```

### 套装款额外计算
```
套装毛利率 = (套装售价 - 单件成本×数量) / 套装售价
套装价 = 单件价 × 倍率（2件套×1.6-1.8 / 5件套×3.5-4.0 / 礼盒装×5.0-6.0）
```

**头程运费估算**：按重量分档
- ≤50g：约 ¥3-4
- 51-100g：约 ¥5-7
- 101-200g：约 ¥8-12
