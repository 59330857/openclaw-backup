# GMV MAX 投流底层逻辑 & 策略指南

> 来源：跨境电商 ROI & 广告跑量核心课程（2026-05-26）
> 定位：投流决策、数据分析、执行方向的核心依据

---

## 一、核心公式链

### 1. ROI 公式
```
ROI = GPM ÷ ECPM
```

### 2. GPM 公式
```
GPM = 点击率 × 转化率 × 客单价 × 1000
```

### 3. ECPM 公式
```
ECPM = 出价 × 预估点击率 × 预估转化率 × 1000
```

### 4. 简化推导（GMV MAX 体系中）
```
ROI = 客单价 ÷ 出价
```

**结论：** 在 GMV MAX 体系里，出价约等于目标 ROI 的倒数。ROI 的上限由客单价决定。

---

## 二、关键指标定义

| 指标 | 含义 | 影响因素 |
|------|------|----------|
| **ECPM** | 千次展示收益，决定平台给量意愿 | 出价、点击率、转化率 |
| **GPM** | 千次曝光成交金额，决定预估排名 | 点击率、转化率、客单价 |
| **ROI** | 投资回报率，最终目标 | 客单价、出价 |
| **点击率** | 被点击的比例 | 素材吸引力、定向精准度 |
| **转化率** | 点击后成交的比例 | 素材内容、产品卖点承接、落地页 |
| **客单价** | 平均订单金额 | 产品力、SKU设计、套餐组合 |

---

## 三、核心结论

### 3.1 出价（ROI）不是唯一变量
内容、产品不过关，出价（ROI）再高也跑不动量。系统看的是 **GPM**，不是出价本身。

### 3.2 GPM 决定拿量能力
- **GPM 高 → 预估排名高 → ECPM 高 → 拿量能力强**
- 平台会主动给 GPM 高的计划放量，即使出价不高

### 3.3 三个可操控杠杆

| 杠杆 | 操作方向 | 备注 |
|------|----------|------|
| **出价** | 调控 ECPM 门槛 | 出价低 → ROI 目标高 → 前期难跑 |
| **客单价** | 提升 ROI 天花板 | 产品迭代、套餐组合、SKU升级 |
| **点击率×转化率** | 提升 GPM 质量 | 素材打磨、产品卖点优化 |

### 3.4 系统优化优先级
```
GPM > 出价
```
优先优化 GPM（素材+产品），再调出价。

## 四、常见现象诊断

| 现象 | 根因分析 |
|------|----------|
| **ROI 设 2 跑不动** | GPM 太低（素材/产品问题），ECPM 竞争不过同赛道计划 |
| **ROI 设 8 能爆单** | 产品客单价高 + 素材 GPM 高，平台主动给量 |
| **计划越跑越差** | CTR/CVR 衰减 → GPM 下降 → ECPM 排名下跌 |
| **出价拉高还是跑不动** | GPM 已经是瓶颈，核心是优化素材和产品 |

## 五、执行方向 Checklist

### 5.1 每日/每周数据复盘
- [ ] 检查各计划 **GPM 趋势**（平稳/下降/上升）
- [ ] 追踪 **点击率 × 转化率** 变化，定位素材衰减节点
- [ ] 对比 **客单价** 与行业/品类均值，评估提升空间
- [ ] 识别高效计划：**高 GPM + 稳定 ROI → 复制放量**

### 5.2 计划调整决策树
```
计划跑不动？
├── 看 GPM 是否低于赛道均值
│   ├── 是 → 优先优化素材/产品，暂缓调出价
│   └── 否 → 检查出价是否具备竞争力
└── 检查 CTR/CVR 趋势
    ├── 衰减中 → 换素材/迭代内容
    └── 稳定但 GPM 低 → 评估产品力和客单价
```

### 5.3 新品起量节奏
1. **测素材**：先跑低成本素材，收集 CTR/CVR 数据
2. **算 GPM**：素材数据稳定后，计算 GPM 预估
3. **定出价**：根据目标 ROI 和客单价反推可接受出价
4. **追规模**：GPM 稳定的计划逐步加预算
