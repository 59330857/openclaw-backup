## TikTok售价价格带管控（核心筛选标准）

**价格带低于以下标准的品，直接过滤，不采集：**

| 子品类 | 最低TikTok售价 | 备注 |
|--------|----------------|------|
| 发夹（猫爪/弹簧夹/抓夹/蝴蝶结） | **≥50,000 VND（$2+）** | 低价跑量款不做 |
| 发圈/头绳/发绑 | **≥30,000 VND（$1.2+）** | 极致低价款不做 |
| 发簪/高档发饰 | **≥80,000 VND（$3.2+）** | 高客单价精品 |

**为什么设下限？**
- 40,000 VND（$1.6）以下的品：平台扣点30% + 物流15% = $0.72固定成本
- 剩下$0.88要cover采购+仓储+退货，根本不够赚
- **低于下限的品，哪怕增长率100%也不跟，因为卖越多亏越多**

**理想黑马价位：80,000-200,000+ VND（$3-$10）**

## 核心原则：毛利率必须≥70%

费用结构拆解（越南）：
- 平台扣点（含佣金+服务费）：~30%
- 物流费（含越南跨境）：~10-15%
- 仓储/人工/货损：~5%
- 退货处理：~3-5%

**毛利率门槛：≥70%**

**趋势黑马拦截 + 三级漏斗验证 + 1688利润终选**

## 四级漏斗式调用原则

### 阶段1：TikTok发饰黑马抓取
**每日额度≤300次 | 每3小时轮询一次（9/12/15/18/21点）**

- 调用 `tiktok_product_trend` 批量接口，监控【时尚配件>发饰】越南站点
- 筛选标准：
  - 销量门槛：24小时销量环比≥100%且连续2天销量正增长
  - 内容质量：近24小时新增带货视频数≥8个
  - 口碑：商品平均星级≥4.2，近7天差评率≤5%
  - 价格门槛：
    - **发夹**：≥50,000 VND / ≥$2
    - **普通发饰**：≥30,000 VND / ≥$1.2
    - **发簪/高档发饰**：≥80,000 VND / ≥$3.2
  - 竞争门槛：同款商品卖家数≤8个
  - 排除：侵权品、退货率≥10%

- **S级紧急爆款**：12小时销量翻倍+视频≥15个+星级≥4.3 → 标记S级，10分钟内完成全流程验证

### 阶段2：Shopee平台生命周期验证
**每日额度≤200次**

- 调用 `shopee_similar_product` 越南站点用标题关键词搜索同款
- 验证标准：
  - 必过：同款月销≥500单、商品评分≥4.2
  - 拦截：无同款、退货率≥12%、评分<4.0

### 阶段3：1688批量供应链利润核算
**每日额度≤350次**

- 调用 `ali1688_similar_product` 接口
- 供应链要求：义乌/广州源头工厂，MOQ≤20件，交货周期≤3天

- **利润公式**：
  ```
  毛利率 = (TikTok售价 - 1688采购价×汇率 - 物流费 - 平台佣金30%) / TikTok售价
  物流费 = TikTok售价 × 15%
  平台佣金 = TikTok售价 × 30%
  ```

- **目标毛利率≥70%才通过**
