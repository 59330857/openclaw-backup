# TK选品 v8.0 — 越南发饰选品标准操作程序（动态潜力款模式）

> 版本：8.0（2026-06-13）
> 市场：越南 VN
> 品类：**发饰**（主营类目）
> 数据源：Shopee热销 → TikTok验证 → 1688利润 → 人工审核 → 组合利润
> 输出终点：**先发给你人工过审，不直接写飞书**

---

## 一、核心逻辑改变

**旧逻辑（v7.0 已废弃）：扫100个 → 毛利率≥70% 硬卡 → 结果全是C级 → 白耗配额**

**新逻辑（v8.0 动态潜力款）：**
1. Shopee 扫描 → 候选池（宽进，毛利≥40%即可入选）
2. TikTok 验证 → 加权信号
3. 1688 查货源 → 毛利参考
4. **→ 发给你人工审核 ←**
5. 通过 → 备选池 → 1688组合方案 → 利润在后端实现

**原则：先让产品进来，利润靠后端结构实现，不在选品入口卡死。**

---

## 二、筛选漏斗

### 2.1 Shopee 初筛（宽进）

**入选条件（全部满足）：**
- 月销量 ≥ 50 单
- 7天销量 ≥ 月销量 × 15%
- 评分 ≥ 4.0
- 上架时间 ≤ 6个月
- 好评率 ≥ 90%
- **价格底线 ≥ 45,000 VND（≈$1.5，不再卡死 $3.0）**

**加分项（满足任一 → 高潜力候选）：**
- MonthlySalesGrowth ≥ 20%
- SalesCountOf7D ≥ 200

### 2.2 TikTok 验证

用 `tiktok_similar_product` 搜 Shopee 标题关键词。

**有同款在跑 → 标注「TK验证」，重点跟进**
**无同款 → 标注「待验证」，可接受但优先级低**

### 2.3 1688 货源查询

用 `ali1688_similar_product` 查货源和采购价。

**毛利率仅作参考指标，不做硬卡：**
- 毛利率 ≥ 60% → 绿色（优秀）
- 毛利率 40-60% → 黄色（可接受）
- 毛利率 < 40% → 红色（偏低，但可接受进入候选池）

### 2.4 人工审核（新增关键步骤）

**我筛完的结果发给你看，不是直接写飞书。**

给你看的内容包含：
- 产品名称 + 主图链接
- Shopee 销量 / 增长率 / 评分
- TK 验证状态
- 1688 货源 + 采购价 + 毛利率参考
- 你的综合判断备注栏

**你的决策：**
- ✅ 通过 → 进备选池
- ❌ 淘汰 → 记原因（下次过滤掉同类）
- ❓ 存疑 → 标注，继续观察

## 三、潜力评分模型（加权评分）

输出「潜力分」供排序参考，满分100：

| 指标 | 权重 | 评分逻辑 |
|------|------|----------|
| 月销量 | 25% | ≥5000→100分，≥1000→80分，≥500→60分，≥200→40分，≥50→20分 |
| 7天增长率 | 25% | ≥50%→100分，≥30%→80分，≥20%→60分，≥10%→40分，≥0%→20分 |
| TK验证 | 20% | 有同款→100分，无→50分 |
| 评分 | 15% | 5.0→100分，4.5→70分，4.0→40分 |
| 上架时间 | 15% | <1月→100分，<3月→70分，<6月→40分 |

**潜力分 ≥ 70 → A级候选**
**潜力分 50-70 → B级候选**
**潜力分 < 50 → C级候选（观望）**

## 四、利润实现路径（后端结构）

毛利兑现不靠单品定价，靠：

1. **拓品**：同方向多款进货，摊低物流/仓储成本
2. **组合套装**：发夹+发绳+发带打包，抬客单价
3. **SKU改造**：颜色/尺寸/包装量变体扩展
4. **互补品搭配**：发饰+发护理品，提升整单利润

## 六、铁律（5条）

| # | 铁律 | 原因 |
|---|------|------|
| 1 | **毛利≥40% 才可入选候选池**（不是 70%），先让产品进来 | 70%硬卡把潜力款全拦了 |
| 2 | **结果必须发给你人工审核，不直接写飞书** | 人工把关，避免系统误判 |
| 3 | **先看类目趋势，再扫产品** | 方向错，扫越多越浪费配额 |
| 4 | **每次写入前查重（按 product_id）** | 防止同批重复写入 |
| 5 | **0产出也必须汇报** | 闷头跑=浪费配额不反馈 |

## 七、已知限制

1. 越南 TK 带货视频数据基本为空，不用视频数作为筛选条件
2. API 连续调用过快报 Auth → 等3秒重试
3. 越南发饰价格带偏低（主流 VND 9,000-49,000），单品毛利空间有限
4. **利润靠后端结构实现**：组合、拓品、SKU改造才是越南发饰的正确打法

## 八、执行流程（每次扫描）

1. Shopee 扫描越南发饰类目（nodeId=11035861）
2. 按潜力评分排序，取前20个候选
3. TikTok 验证每个候选
4. 1688 查货源和毛利率参考
5. 生成候选池报告，发给你人工审核
6. 你审核通过 → 进备选池
7. 备选池满5个 → 输出组合方案
8. 汇报（扫描数量 / 候选数量 / 你的审核结果 / 配额消耗）
