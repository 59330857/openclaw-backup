## 一、精准流量词 → TK精准标签词（一切分析的基础）

### 亚马逊逻辑
精准词 = 能精准描述功能、材质、特征、场景、对象的词

### TK东南亚适配

| 流量词类型 | 亚马逊 | TikTok Shop |
|-----------|--------|-------------|
| 核心词 | 搜索关键词 | Hashtag标签（越南语/泰语） |
| 长尾词 | 精准属性词 | 场景词 + 人群词 + 用途词 |
| 找竞品方式 | 关键词反查 | **达人视频标签反查** |

### 精准标签词的作用（TK版本）
1. **找到直接竞品** → 做市场/竞品分析
2. **广告架构简单** → TK内容种草+小店闭环，转化有保障
3. **反查流量成本** → 算短视频CPM/带货GMV投产比
4. **判断真蓝海**：
   - 小众 + 有精准标签 + 弱竞品 = **真蓝海**
   - 小众 + 无精准标签 = 没市场

### 标签词获取方法
- EchoTik商品搜索 → 看关联标签
- TikTok搜索 `$关键词 + 带货` → 看达人视频标签
- 越南语/泰语本地化精准词（**必须本地语言！**）

## 二、核心参数卡死（源头过滤红海）—— 亚马逊精铺精髓

> 亚马逊精铺第一步：用固定参数筛红海，从源头过滤大部分无效品
> TK版本完全适用，参数需根据TK市场特点调整

### 参数设置（直接抄作业）

| 参数 | 保守标准 | 常规标准 | 目的 |
|------|---------|---------|------|
| **月销量** | ≤200 单/月 | ≤500 单/月 | 避开头部扎堆爆品赛道 |
| **评分数**（匹配上架时间） | 近半年上架 ≤30 条 | 近两年上架 ≤100 条 | 低评也能稳转化，运营压力小 |
| **客单价** | ≥30 美金等效 | ≥20 美金等效 | 过滤低价内卷、利润盖不住广告的品 |
| **配送方式** | FBA+FBM 全渠道 | FBM 为主 | 全渠道竞品不遗漏（TK以本地发货为主） |
| **季节匹配** | 未来 3-4 个月为旺季 | 未来 2-3 个月为旺季 | **时光机法则：提前布局，接住流量爆发** |

### 时光机法则（关键！亚马逊精髓）
```
4月选 8月旺季 → 货刚好赶上流量爆发
4月选 9月旺季 → 国庆节/圣诞提前备货
```

### 参数卡死落地工具
- **亚马逊**：卖家精灵「选产品」→ 直接照抄参数
- **TK版本**：EchoTik 筛选器 + 手动参数核对
  - 月销量 → EchoTik 商品列表「月销量」排序
  - 评分数 → 商品详情页「评论数」
  - 上架时间 → 商品详情页「上架时间」

## 三、三步快速验证（亚马逊精铺核心 → TK适配版）—— ★ v5.0 新增核心

> **亚马逊方法论**：精准词验可行性 → 竞价算成本 → 保守测盈利
> **核心逻辑**：在深入分析前，用最少步骤判断这个品值不值得做

### 第一步：查精准属性词（判断转化潜力）

**亚马逊逻辑**：找销量最好的直接竞品 → H10 反查关键词 → 无精准词直接 PASS

**TK版本**：
```
步骤：
1. 在 EchoTik 找到目标品类销量最好的直接竞品
2. 反查其关联的 Hashtag 标签 / 达人视频标签
3. 判断：有精准标签 → 好转化、易运营；无精准标签 → 直接 PASS

判断标准：
- 有 3+ 个精准标签（越南语/泰语）→ 转化有保障
- 精准标签与目标人群匹配 → 内容种草友好
- 无精准标签或标签模糊 → 市场教育成本高，慎入
```

### 第二步：查关键词实时竞价（算广告/流量成本）

**亚马逊逻辑**：用无出单记录的低权重竞品反查，数据更保守

**TK版本**：
```
步骤：
1. 找到低权重竞品（0基础视频、无粉丝账号也能出单的品）
2. 反查其短视频 CPM / 达人佣金比例
3. 估算流量成本对盈利底线的影响

关键指标：
- 短视频 CPM（千次播放成本）：判断内容投放性价比
- 达人佣金比例：10-20% 正常，超过 25% 压缩利润
- 自然流量占比：自然流量占比高 → 广告依赖低 → 更健康
```

### 第三步：保守测算盈利（决定最终是否深入）

**亚马逊逻辑**：假设转化率 5%-7%、无自然流量 → 用选品利润表计算

**TK版本（完全沿用）**：
```
假设条件（保守）：
- 转化率：5%-7%（参考同类爆款）
- 自然流量：暂不计，自然流量是额外安全垫
- 达人佣金：10%（可上下浮动）

计算公式：
真实毛利润 = 售价_RMB - 进货价 - 物流费 - 总平台成本（41%+¥2）
真实毛利率 = 真实毛利润 / 售价_RMB

结论判断：
- 毛利率 ≥ 30%（单件）/ ≥ 45%（套装）→ 盈利，继续做竞品分析
- 毛利率 20-30% → 微亏，需结合其他维度综合判断
- 毛利率 < 20% → 直接放弃
```

## 四、投产比与资金效率

### 标准（与亚马逊一致）
**投产比 ≥ 120%**

### TK版投产比计算
```
短视频投产比 = (GMV - 成本) / 投放成本 × 100%
自然流量投产比 = GMV / 运营成本 × 100%
```

### 看两点
1. **市场健康度**：大量竞品投产差 = 市场不健康
2. **资金效率**：TK回款周期比亚马逊快，但**高周转**仍是关键

### 越南站利润模型（v3.0）
```
汇率：3570 VND/USD
平台费率：13% + 5% + 12% + 1% + 10% = 41% + ¥2固定

售价_RMB = 售价_VND / 3570
总平台成本 = 售价_RMB × 41% + 2
真实毛利润 = 售价_RMB - 进货价 - 物流费 - 总平台成本
真实毛利率 = 真实毛利润 / 售价_RMB

毛利率要求：
- 单件 ≥ 20%
- 套装款 ≥ 40%
```

### 泰国站
```
售价 (THB) = (1688采购价CNY + 头程运费CNY) × 5.0 × 1.25
毛利率要求：≥ 25%
```

## 十、备货黄金法则（亚马逊精髓 → TK适配）—— ★ v5.0 新增核心

> 亚马逊：市场月体量≈1000单、头部月销≈200单 → 首单80-100件，目标月销50单
> TK逻辑：体量更小、更分散，首单应更保守

### TK发饰备货法则

```
判断基准：
- 市场月体量 ≈ 500-800 单（越南）/ 200-400 单（泰国）
- 头部月销 ≈ 100-200 单（越南）/ 50-100 单（泰国）

首单计算（保守法则）：
首单备货量 = EchoTik 热销款月销量 × 20-30%
上限：不超过 100 件
下限：不低于 30 件（满足变体测试需求）

目标月销：30-50 单（越南）/ 20-30 单（泰国）
```

### 备货决策矩阵

| 市场信号 | 首单策略 |
|---------|---------|
| 热销款月销 100-200 单 | 首单 30-50 件，测试 2 周 |
| 热销款月销 200-500 单 | 首单 50-80 件，测试 3 周 |
| 热销款月销 500+ 单 | 首单 80-100 件，测试 4 周 |
| 新品/无数据 | 首单 30 件，快速测款 |

### 补货法则
```
2周内动销 ≥ 30% → 立即补货（追单）
2周内动销 15-30% → 正常补货
2周内动销 < 15% → 停止补货，下架处理
```

## 十一、发货数量 + 变体循环布局

### 亚马逊逻辑
- 首批备货：参考≤30评竞品平均销量的50%
- 变体布局：引流变体 → 销售变体 → 备用变体 → 补货变体
- 分散库存，防差评影响全链接

### TK发饰版本（变体堆叠是杀手锏！）

| 变体维度 | 可扩展数量 | 效果 |
|---------|-----------|------|
| 颜色变体 | 5-10 色 | 评论共享，5个颜色=1个链接 |
| 规格变体 | 3+ 种 | S/M/L 多规格覆盖 |
| 套装形式 | 2+ 种 | 2件套/5件套/节日礼盒 |

### 变体堆叠落地策略
```
一次性上架所有颜色变体 → 评论全部累计共享
↓
5个颜色只有1个链接 → 评价积累速度 x5
↓
搜索权重集中 → 自然流量集中
↓
降低广告依赖 → 利润率提升
```

### 变体布局时序
1. **测试变体**：先上3-5个颜色测试
2. **爆量变体加码**：哪颜色出单快，快速补充库存
3. **滞销变体淘汰**：2周无动销则下架
4. **节日变体前置**：节日前45天准备好特殊颜色/包装

## 十二、评估新品短视频种草友好度（新品好不好起）

### 亚马逊逻辑
看新品（≤30评）前5销量 ÷ 头部前5销量 > 20%

### TK版本：看短视频冷启动能力

| 信号 | 含义 |
|------|------|
| **0基础视频也能跑出量** | 市场需求旺，内容驱动有效 |
| **无粉丝账号发视频也能出单** | 对新手友好，流量分配去中心化 |
| **低评分也能卖** | 市场不饱和，评价体系尚未固化 |
| **FBM/Shipping也稳定出单** | 物流不是核心瓶颈 |

### TK核心冷启动指标
- **新品视频发出去48h内自然播放量**
- **0基础账号发视频的带货转化率**
- **同类视频平均GMV vs 新品GMV占比**

### 判断标准
```
新品短视频种草系数 = 新品48h播放量 / 同类平均播放量
> 30% = 对新品友好，市场机会大
< 15% = 新品难以起量，慎入
```

## 选品 SOP 完整流程（v5.0 新增）

```
第一步：参数卡死（5分钟）
├── 月销量 ≤ 500（越南）/ ≤ 300（泰国）
├── 评分数 ≤ 100（近2年上架）
├── 客单价 ≥ 20 美金等效
└── 季节匹配：未来 3-4 个月为旺季

第二步：三步快速验证（10分钟）
├── 精准词验证：有 3+ 个精准标签 ✓
├── 竞价/CPM 合理：佣金 ≤ 25%，CPM 可接受 ✓
└── 保守盈利：毛利率 ≥ 25%（单件）/ ≥ 45%（套装）✓

第三步：通用词延伸（5分钟）
├── 按场景/节日/人群延伸 2-3 个细分变体
└── 验证延伸款销量是否达标

第四步：差异化 + 供应链确认（5分钟）
├── 变体扩展性：颜色≥5 或 规格≥3
├── 供应商：评分≥4.5，发货≤48h
└── 合规检查：无 IP 侵权

第五步：评分 + 决策（5分钟）
├── 8维度评分 ≥ 100 分
├── 毛利率达标
└── 三步验证全部通过

→ 通过 → 写入飞书 Bitable → 采购采集
→ 不通过 → 记录放弃原因 → 下一款
```

## 行为准则

1. **数据真实性**：绝不伪造数据，暂时无法获取的指标标记为"暂无数据"
2. **IP零容忍**：发现任何IP侵权风险商品，无论利润多高，一律排除
3. **利润优先**：毛利率不达标（泰<25%/越<20%）的商品一律不推荐
4. **及时更新**：当商品出现差评率上升、竞争加剧等信号时，主动下调等级或移出推荐池
5. **中文输出**：所有报告、通知、摘要均使用中文
6. **三步验证前置**：未通过三步验证的商品不做深入分析
7. **保守测算是底线**：假设无自然流量，盈利才能做，不盈利坚决不做
