**使用场景**
- 快速生成 5‑10 条候选 SKU 列表  
- 对比供应商报价、成本、预计利润  
- 输出选品评估表（Markdown / Feishu Bitable）  

**基本指令**
- `list-candidates` – 列出当前候选产品（名称、品类、来源、预估利润）。
- `add-candidate <name> <category> <supplier_url> <cost> <price>` – 手动加入新候选。
- `evaluate <name>` – 依据已有数据计算利润率、竞争指数并给出建议。
- `export-bitable` – 将当前选品库导出至 Feishu 多维表格（需要后续配置 Bitable App）。
- `clear` – 清空所有候选记录（慎用）。

**后续扩展（留白）**
- 接入爬虫/Marketplace API（AliExpress、Amazon、eBay 等）自动抓取热销数据。  
- 结合 TikTok Trending API 过滤符合平台热点的商品。  
- 自动生成产品卖点脚本、短视频脚本草稿。  

**注意事项**
- 选品决策请结合实际供应链、海关政策与当地法规。  
- 数据仅作参考，实际利润需经过财务复核。

执行后 `clawhub` 会把对应的 Skill 拉取到 `~/.openclaw/extensions/` 目录下并自动生成必要的 `openclaw.plugin.json`，随后你就可以在 `openclaw skill run <skill-name> …` 中直接调用它。
